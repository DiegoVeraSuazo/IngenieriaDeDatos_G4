Se abordarn los dos problemas propuestos de manera similar, en el caso del problema de clasificacin de la columna NOM_CAT_ENT se crear un modelo de prediccin. Para entrenar los rboles de decisin, cada vez se separar el dataset que se utilizara en un conjunto de entrenamiento y uno de prueba, que se dividiran en la proporcin 70/30 respectivamente ya que estos son porcentajes generalmente buenos segn lo visto en clases y no se considera que haya algn motivo en el contexto para usar otra proporcin.

Luego de entrenar los modelos se observar si alguno de los modelos entrenados con los dataset 1, 2 o 3 tienen mejor precisin que el modelo entrenado con el dataset 4. Si es asi se realizar un entrenamiento adicional, se entrenar un rbol de decisin de la misma manera pero se utilizara un dataset que combine el dataset que obtuvo mejor precisin con el dataset (distinto del 4) que haya obtenido la segunda mejor precisin.
